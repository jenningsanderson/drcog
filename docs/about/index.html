Colorado Building Import

What is the DRCOG Import?

The Denver Regional Council of Governments has purchased building footprint data for much of the Front Range of Colorado. We are now hoping to import all of these building footprints into the free and open map of the world, OpenStreetMap.

What is an Import-a-thon?

There are over 1M building footprints to be imported. These buildings need to be reviewed
