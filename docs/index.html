The Denver Regional Council of Governments (DRCOG) has purchased over 1M building footprints for the greater Denver area. We're going to add them all to OpenStreetMap!

OpenStreetMap is an open map of the world that anyone can edit and use freely. In support of open data, The Denver Regional Council of Governments is making over 1M building footprints with 6cm accuracy available for importing to OpenStreetMap.

Over the next few months, various import-a-thons across the Colorado Front-Range will make this region one of the best-mapped in OpenStreetMap!

What is an Import-a-thon?

There are over 1M building footprints to be imported. These footprints, like the ones shown here, need to be reviewed by volunteers for accuracy and attributes. When merging data into OSM, we need to compare to any existing data and conflate attributes properly!
